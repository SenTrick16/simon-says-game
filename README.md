#  Simon Says Game

A browser-based **Simon Says** memory game built with vanilla HTML, CSS, and JavaScript.

##  Live Demo
sentrick16.github.io/simon-says-game/

##  How to Play
1. Press any key to start the game
2. Watch the sequence of colored buttons that light up
3. Repeat the sequence by clicking the buttons in the same order
4. Each level adds one more step to the sequence
5. Game over if you press the wrong button!

##  Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

##  Features
- Progressive difficulty (sequence grows each level)
- Visual flash feedback for both game and user input
- Score display on game over
- Keyboard trigger to start/restart
